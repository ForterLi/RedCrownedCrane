# RedCrownedCrane
Local databases can be synchronized to iCloud using Sqlite Realm WCDB


### Installation

#### CocoaPods

```ruby
source 'https://github.com/CocoaPods/Specs.git'
platform :ios, '12.0'
use_frameworks!

target 'MyApp' do
  pod 'RedCrownedCrane', '~> 1.0.0'
end
```
